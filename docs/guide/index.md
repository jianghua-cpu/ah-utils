# 介绍

ah-utils-tools 是一个轻量级的前端工具库，专注于为现代前端开发提供简洁、高效、类型安全的工具函数。

## 特性

- **零依赖** - 不依赖任何外部库，保持最小 bundle 体积
- **类型安全** - 完整的 TypeScript 类型定义
- **Tree-shaking** - 支持按需导入，自动排除未使用的代码
- **多环境** - 支持浏览器、Node.js、微信小程序等环境
- **多格式** - 提供 ESM、CJS、UMD 三种导出格式

## 模块概览

| 模块 | 描述 |
|------|------|
| [类型判断](/api/type) | 判断各种数据类型，如字符串、数字、数组等 |
| [字符串](/api/string) | 字符串处理，如去空格、格式化、命名转换等 |
| [数组](/api/array) | 数组操作，如去重、分块、交集、并集等 |
| [对象](/api/object) | 对象操作，如深拷贝、属性获取、合并等 |
| [浏览器](/api/browser) | 浏览器相关工具，如防抖、节流、存储等 |

## 为什么选择 ah-utils-tools？

相比 lodash 等大型工具库，ah-utils-tools 专注于提供最常用的功能，体积更小、加载更快。同时，我们保持 API 的简洁性和一致性，让开发者能够快速上手。
